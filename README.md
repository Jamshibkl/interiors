# Kaveri Interiors

Marketing website for **Kaveri Interiors** — a boho-modern interior design studio in Hyderabad.
Static, multi-page site (HTML/CSS/JS, no framework) with an optional Node/Express backend for form submissions.

## ✨ Features

- Responsive boho-modern design with a warm sand / terracotta / charcoal palette
- Intro preloader, custom cursor, scroll animations, animated founder section
- **3D coverflow** room explorer in the gallery
- Filterable, photo-driven service & workshop pages
- 10-step process timeline, testimonials, FAQ, newsletter
- Social links (Instagram, Facebook, YouTube, Pinterest, Threads) + WhatsApp float
- Quote / contact forms (graceful WhatsApp fallback when no backend)

## 🛠 Editing

- **Home page** → edit `index.html` directly.
- **Inner pages** (about, services, workshop, gallery, careers, contact) are
  generated. Edit their content in `scripts/build-pages.js`, then rebuild:

  ```bash
  npm run build        # = node scripts/build-pages.js
  ```

- **Styles** → `css/styles.css`. **Behaviour** → `js/app.js`.

## 💻 Run locally

Any static server works. Two easy options:

```bash
# 1) Plain static (no backend) — forms fall back to WhatsApp
npx serve .            # or: python -m http.server 8000

# 2) With the Node form backend
npm install
npm start              # http://localhost:3000  (saves form posts to submissions.json)
```

## 🚀 Deploy

This is a static site — host the repo root as-is.

- **GitHub Pages:** push to GitHub → Settings → Pages → Deploy from branch → `main` / root.
- **Netlify / Vercel / Cloudflare Pages:** import the repo, build command: *none*,
  publish directory: `/` (root).

The contact/quote forms POST to `/api/*`. On a static host those endpoints don't
exist, so the site automatically falls back to opening WhatsApp. To capture form
submissions, deploy `scripts/server.js` to a Node host (Render, Railway, etc.).

## 🔧 Configure before launch

- WhatsApp number — `WA_NUMBER` in `js/app.js` and the `wa.me/…` links in
  `index.html` + `scripts/build-pages.js`.
- Phone / email — top bar, footer and contact page (search `00000` / `hello@kaveriinteriors.com`).
- Facebook URL — the `'#'` placeholder in the `SOCIALS` list in `js/app.js`.
- Project tour video — `YOUTUBE_ID` in `js/app.js`.
